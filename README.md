# About
Daily Checker is a Flutter app for Android that helps people record their daily activities and if they are completed or not.


## Features
- [x] Everyday activity check list
- [x] Past days history viewer
- [x] 3 configurable reminders
- [x] Support to 9 languages
- [x] A few date format options
- [x] A few color themes options
- [x] Adaptative design for bigger screens (like tablets)


## Supported Languages 
 - [x] English
 - [x] Portuguese
 - [x] Brazilian Portuguese
 - [x] Spanish
 - [x] Italian
 - [x] German
 - [x] Japanese
 - [x] Chinese (Simplified)

    flutter pub get

Start the app:

    flutter run
