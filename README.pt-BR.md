# Sobre
Daily Checker é um aplicativo Flutter para Android que ajuda as pessoas a registrar suas atividades diárias e se elas foram concluídas ou não.


## Recursos
- [x] Lista de verificação de atividades diárias
- [x] Visualizador de histórico de dias anteriores
- [x] 3 lembretes configuráveis
- [x] Suporte para 9 idiomas
- [x] Algumas opções de formato de data
- [x] Algumas opções de temas de cores
- [x] Design adaptável para telas maiores (como tablets)

## Idiomas suportados
- [x] Inglês
- [x] Português
- [x] Português do Brasil
- [x] Espanhol
- [x] Italiano
- [x] Alemão
- [x] Japonês
- [x] Chinês (simplificado)

## Desenvolvimento
Faça um fork/clone do repositório, depois instale as dependências:

    flutter pub get

Inicie o aplicativo:

    flutter run
